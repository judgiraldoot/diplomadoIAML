Regresión Lineal¶

Introducción¶

La regresi√≥n es una t√©cnica de an√°lisis estad√≠stico com√∫nmente utilizada para la inferencia, predicci√≥n y construcci√≥n de modelos estad√≠sticos que permite describir el comportamiento de una poblaci√≥n siempre y cuando la muestra cumpla con un conjunto de supuestos como, por ejemplo, la normalidad y homocedasticidad de los errores y la independencia de las mediciones. Como modelo estad√≠stico permite plantear un conjunto de ecuaciones matem√°ticas a partir de las cuales se pueden estudiar las relaciones funcionales entre diferentes propiedades o caracter√≠sticas de la poblaci√≥n o sistema analizado. Como herramienta de inferencia permite hallar los mejores estimadores puntuales de los par√°metros del sistema, representados en los coeficientes de las ecuaciones matem√°ticas, junto a sus intervalos de confianza. En cuanto a herramienta de predicci√≥n permite obtener una herramienta √∫til para obtener informaci√≥n sobre nuevas observaciones, escenarios o configuraciones no estudiadas con anterioridad y el error asociado a esta predicci√≥n.

Para dar una leve observaci√≥n a la herramienta de regresi√≥n lineal, se observar√° el planteamiento del modelo de regresi√≥n desde el modelo matem√°tico, pasando por el de un problema de estudio por el planteamiento de un algoritmo en Python.

Historia¶

En 1805 Legendre public√≥ la primera forma de regresi√≥n lineal basada en el m√©todo de los m√≠nimos cuadrados para posteriormente ser complementado por Gauss en 1821. En un principio la regresi√≥n se utiliz√≥ por primera vez en el estudio de variables antopom√©tricas con el fin de comparar la estatura de padres e hijos. En donde se concluy√≥ que aquellos hijos con padres muy bajos tend√≠an a reducir su diferencia con respecto a la estatura media, es decir, "regresaban" al promedio. El t√©rmino lineal es utilizado para distinguir la t√©cnica del resto de regresi√≥n que emplean modelos basados en funciones matem√°ticas.

Modelo¶

Estad√≠sticamente el modelo de regresi√≥n lineal consiste en ajustar los datos a una l√≠nea recta por medio de la ecuaci√≥n $y_i=\beta_0+\beta_1 x_i + \varepsilon$ donde $y_i$ es una variable dependiente de $x_i$ y $\varepsilon_i$ el error de predicci√≥n de cada una de las observaciones. Para esto, com√∫nmente se aplica el m√©todo de optimizaci√≥n de m√≠nimos cuadrados sobre la funci√≥n de p√©rdida definida como $$ ECM(\beta_0, \beta_1)= \sum_{i=1}^n ||y_i -\beta_0-\beta_1 x_i||^2.$$ Una soluci√≥n est√°tica a este problema es el conjunto de ecuaciones conocidas como ecuaciones normales

$$ \hat{\beta}_0 = \dfrac{\sum_{i=1}^n x_i^2\sum_{i=1}^n y_i - \sum_{i=1}^n x_i y_i \sum_{i=1}^n x_i}{n\sum_{i=1}^n x_i^2 - \left(\sum_{i=1}^n x_i \right)^2} = \bar{y}-\beta_1\bar{x},$$$$ \hat{\beta}_1 = \dfrac{\sum_{i=1}^n x_i y_i - n \bar{x}\bar{y}}{\sum_{i=1}^n x_i^2 - \dfrac{1}{n} \left(\sum_{i=1}^n x_i \right)^2}. $$

Sin embargo, en la actualidad hay un gran conjunto de variaciones de funciones objetivo enfocadas en resolver problemas espec√≠ficos como el sobre ajuste de los datos, las cuales suelen ser resueltos por medio de algoritmos de optimizaci√≥n iterativos, como el gradiente descendiente estoc√°stico.

Problema de aplicación - Base de datos de Baependi¶

Para el problema de aplicaci√≥n se utilizar√° la base de datos sobre problemas cardiovasculares realizada en la poblaci√≥n de Baependi Brasil en el 2005 y actualizada cada 5 a√±os a partir de la fecha. La base de datos cuenta con las variables edad, rango edad, genero, peso, altura, glucosa, colesterol de alta densidad (CHDL), colesterol de baja densidad (CLDL), trigliceridos, indice de masa corporal (IMC), media de presi√≥n arterial sist√≥lica (mediaPAS), media de presi√≥n arterial diast√≥lica (mediaPAD) y urea.

# Creamos el objeto de Regresi√≥n Linear
regr = linear_model.LinearRegression()
 
# Entrenamos nuestro modelo
regr.fit(X_train, y_train)
 
# Hacemos las predicciones que en definitiva una l√≠nea (en este caso, al ser 2D)
y_pred = regr.predict(X_train)
 
# Veamos los coeficienetes obtenidos, En nuestro caso, ser√°n la Tangente
print('Coefficients: \n', regr.coef_)
# Este es el valor donde corta el eje Y (en X=0)
print('Independent term: \n', regr.intercept_)
# Error Cuadrado Medio
print("Mean squared error: %.2f" % mean_squared_error(y_train, y_pred))
# Puntaje de Varianza. El mejor puntaje es un 1.0
print('Variance score: %.2f' % r2_score(y_train, y_pred))
